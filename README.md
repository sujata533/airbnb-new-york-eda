# New York Airbnb Listings EDA

This project analyzes Airbnb listings in New York to understand pricing patterns, neighborhood demand, and host activity.

## Dataset
- Source: Kaggle
- Records: ~20,000 listings

## Features include:
- price
- room_type
- neighbourhood_group
- number_of_reviews
- availability_365
- rating

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Manhattan has the highest average prices.
- Entire homes are more expensive than private rooms.
- Brooklyn and Manhattan have the highest number of listings.
